机床功能介绍
参数备份恢复

具体参数设置请参考参数备份恢复界面

加工
  1. 打开备份恢复界面，将模式选择设置为加工
  2. 启动程序
备份
  1. 打开备份恢复界面，将模式选择设置为备份
  2. 设置备份路径、备份目录、文件名称
  3. 如需备份到前置USB，需要将配置文件 extdev.ini 放入系统目录: /oem/sinumerik/nck，并重启系统。测试无法备份到后置 USB
  4. extdev.ini 文件内容如下：
    5. ;****************************
    ;备份时写入文件地址定义文件
    ;放入 oem 或 user 路径下, user 优先级高于 oem. /oem/sinumerik/nck/extdev.ini
    ;修改配置文件后需要重启 nck 才会生效
    ;测试只能识别前置USB接口, 后置 DEV_3 无效
    [ExternalDevices]

    ;/dev/ext/8 = "//DEV_3, /, O"
    /dev/ext/9 = "USB, /, O"
    ;****************************
  5. 根据需要设置备份模式
  6. 如果设置为自定义变量模式则需要定义：列表目录、列表名称、列表行数
  7. 变量列表路径默认在子程序目录下
  8. 变量列表文件定义规则如下：
    1. 参数最多不可超过800个
    2. 需要定义参数文件总行数，包括分号注释所占行和空行
    3. 参数文件内可在行开头用分号 " ; " 定义注释，注释所在行不会被读取，空行也不会被读取
    4. 参数名称前加星号 "*" 表示字符串变量
  9. 示例自定义备份变量表文件内容如下：
    10. ;****************************
    ;PAR_DEMO.TXT

    ;此文件为示例参数列表文件
    ;需要在行数参数中声明此文件总共有多少行,此示例文件需要定义的行数为 14
    ;参数文件内可在行开头用分号 " ; " 定义注释，注释所在行不会被读取，空行也不会被读取
    ;参数名称前加星号 "*" 表示字符串变量

    R0
    R1

    INI[5]
    *STRING
    ;注释行
    WORK[5]
    ;*****************************
恢复
  1. 打开备份恢复界面，将模式选择设置为恢复
  2. 设置恢复路径、恢复目录、文件名称
  3. 如需恢复存储在后置USB的文件，需要在逻辑驱动器中配置好，名称设置为：//DEV_3
注意事项
  1. 自定义备份变量表文件需要定义在系统子程序路径下
